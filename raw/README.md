# Taranga+ Preview

## Ki甸ে দেখতে পারবে?
Ei folder diye tumi Taranga+ app ta preview korte parbi.

## 2 Ta Upay App Dekhte:

### Method 1: Static Preview (Recommended)
1. `preview.html` file ta double-click koro
2. Browser e app ta khulbe
3. 551 channels grid e dekhte pabi

### Method 2: Full Dev Server
1. `run-app.bat` file ta double-click koro  
2. Terminal khulbe
3. Browser e `http://localhost:3000` jao
4. Full app ta chalbe!
